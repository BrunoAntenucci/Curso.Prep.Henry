Una variable sirve para almacenar datos. Los datos pueden ser de varios tipos: numeros, cadenas de caracteres y operadores logicos son los mas comunes. Las variables se pueden modificar en JS salvo las que se declaran usando el comando const, en ese caso son constantes y no se pueden modificar.
Las strings son un tipo de dato que concatena caracteres. En JS se ponen entre comillas
Las funciones son formas de hacer operaciones con datos, que ya pueden estar incluidas en JS o las puede crear el mismo programador para agregar una funcionalidad que JS no tiene. Las funciones aceptan argunmentos, que son justamente los datos sobre los que va a operar. Lo que hace una funcion dentro de si misma no escapa de los limites de la funcion, una variable definida dentro de una funcion, quedaria sin definir por fuera de la misma. A esto se le llama scope o alcence. La forma de presentar datos por fuera de la funcion es con return.
Las declaraciones if son condicionales: si se cumple las condiciones que pide el if, entonces uno puede programar para que se lleven a cabo alguno operacion, si en cambio no se cumplen, se puede hacer que se opere de otra forma.
Los valores booleanos determinan un valor de verdad, por ejemplo si las condiciones de un if se cumplen (true) o no (false)
